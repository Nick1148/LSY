# 찐테스트 고급화 계획 (Premium UI Upgrade Plan)

## 현재 문제점 요약

1. **배터리 테스트가 파스텔톤과 안 맞음** - 배터리 UI가 딱딱하고 기계적, 파스텔 감성과 괴리
2. **테스트 간 시각적 통일감 부족** - 뇌탭 vs 배터리가 다른 사이트처럼 느껴짐
3. **바이럴 요소 부족** - 결과 화면이 캡처/공유하고 싶을 만큼 예쁘지 않음
4. **랜딩 페이지 전환 경험 부족** - 메인→테스트 이동 시 연결감 없음

---

## Phase 1: 사회적 배터리 파스텔 리디자인 ✅

### 1-1. 배터리 UI를 "귀여운 배터리 캐릭터"로 변경 ✅
- ✅ 현재: 각진 직사각형 배터리 → **둥근 알약형 배터리**로 변경
- ✅ 배터리 테두리: 4px 단색 → **그라데이션 보더 + 글로우 이펙트**
- ✅ 배터리 캡: 작은 직사각형 → **귀여운 둥근 캡 + 반짝임 이모지**
- ✅ 배경에 작은 번개⚡/하트💖 파티클 떠다니게

### 1-2. 배터리 색상 파스텔화 ✅
```
현재                    →  변경
100%: #6EE7B7 (민트)    →  #B5EAD7 (소프트 민트) + 반짝이 효과
80%:  #86EFAC (연두)    →  #C7CEEA (라벤더 블루)
60%:  #FDE68A (노랑)    →  #FFDAC1 (피치)
40%:  #FDBA74 (오렌지)  →  #FFB7B2 (코랄 핑크)
20%:  #FCA5A5 (빨강)    →  #E2B6CF (더스티 로즈)
0%:   #F9A8D4 (핑크)    →  #D5AAFF (라벤더)
```

### 1-3. 결과 카드 프리미엄화 ✅
- ✅ 결과 배경: 단색 → **파스텔 그라데이션 + 별 일러스트**
- ✅ 캐릭터 이모지 크기 확대 (3rem → 5rem) + 바운스 애니메이션
- ✅ "충전 방법" 카드: 플랫 → **글래스모피즘 카드**
- ✅ 궁합 섹션: 텍스트 → **아바타 카드 형태** (아이콘 + 레이블 + 타입명)

### 1-4. 질문 화면 개선 ✅
- ✅ 선택지 버튼: **카드형 버튼** (호버 시 떠오름 + 글로우)
- ✅ 선택 시 효과: **confetti 파티클 + 부드러운 스케일업**
- ✅ 프로그레스바: 단색 → **그라데이션 + 충전 아이콘 이동**

## Phase 2: 뇌 탭 테스트 고급화 ✅

### 2-1. 브라우저 윈도우 프리미엄화 ✅
- ✅ 브라우저 상단바: 플랫 → **파스텔 그라데이션 + 그림자**
- ✅ 트래픽 라이트: 기본 원 → **호버 시 귀여운 표정 (😊😴😡)**
- ✅ URL 바: **"brain://my-thoughts/${type}" 귀여운 URL 표시**

### 2-2. 탭 디자인 개선 ✅
- ✅ 탭 모양: 직사각형 → **둥근 모서리 + 파스텔 배경색 6종 랜덤**
- ✅ 탭 등장: 단순 fadeIn → **위에서 떨어지듯 + 통통 튀는 효과**
- ✅ 닫기 버튼: × → **귀여운 ✕ 아이콘 + hover 시 빨개짐**

### 2-3. RAM 시각화 개선 ✅
- ✅ RAM 바: **웨이브 애니메이션 (shimmer sweep)**
- ✅ 퍼센트: 숫자만 → **이모지 표정 변화 (😴→😌→😅→🤯)**
- ✅ 과부하 시: **연기/불꽃 파티클 효과 (💨🔥💥⚡💢)** + 글로우 경고

### 2-4. 결과 화면 통일 ✅
- ✅ 배터리 테스트와 동일한 카드 레이아웃 적용
- ✅ 글래스모피즘 결과 카드 (backdrop-filter blur)
- ✅ 특성 태그: 보더 → **파스텔 배경 칩 (pill shape)**

## Phase 3: 공통 인프라 고급화 ✅

### 3-1. 마이크로 인터랙션 추가 ✅
- ✅ 카드 전환: **fade + slide 트랜지션** (cardIn 애니메이션)
- ✅ 버튼 호버: **살짝 떠오름 + 글로우 + 스케일**
- ✅ 스크롤: **스무스 스크롤**

### 3-2. 공유 기능 고급화 ✅
- ✅ 결과 이미지: **프리미엄 프레임 (상단 그라데이션 라인) + 워터마크**
- ✅ 공유 버튼: **호버 애니메이션 + 복사 시 confetti 폭발**
- ~~인스타 스토리용 세로 비율 캡처 옵션~~ → 제외 (니치 기능, 기본 캡처로 충분)
- ~~QR코드~~ → 제외 (결과 이미지가 과잉 요소로 지저분해짐)

### ~~3-3. 사운드 이펙트~~ → 전체 제외
- 모바일 브라우저 자동재생 제한, 대부분 무음모드 사용
- 사운드가 오히려 사용자 이탈 요인이 될 수 있어 제외

### 3-4. 로딩/계산 화면 개선 ✅
- ✅ 스피너 → **바운스 캐릭터 애니메이션** (이모지가 좌우 흔들리며 바운스)
- ✅ **단계별 메시지** (4단계 + 이모지 캐릭터 변경)
- ✅ **프로그레스 바** (그라데이션 채워짐)

## Phase 4: 랜딩 페이지 & 네비게이션 ✅

### 4-1. 테스트 선택 허브 리디자인 ✅
- ✅ 카드 레이아웃: 리스트형 → **가로 스크롤 (scroll-snap)**
- ✅ 각 카드: 대표 이모지 + **카드별 참여자 수** + "HOT" 뱃지
- ✅ 카드 hover: **3D tilt 효과 (perspective + rotate) + 빛 추적 (radial-gradient)**
- ✅ 스크롤 인디케이터 (dot 네비게이션)

### 4-2. 테스트 간 연결 ✅
- ✅ 결과 화면 하단: "다른 테스트도 해보세요!" CTA + 링크 카드
- ~~추천 로직: 결과 기반 다른 테스트 추천~~ → 제외 (테스트 2개뿐이라 오버엔지니어링)

## Phase 5: 고급 기능 추가 ✅

### 5-1. 다크모드 지원 ✅
- ✅ 토글 버튼: **☀️/🌙 아이콘** (고정 우상단, 글래스모피즘)
- ✅ 파스텔 다크: **딥 네이비 #1A1625 배경 + 밝은 파스텔 텍스트**
- ✅ localStorage 저장 + 시스템 테마 자동 감지
- ✅ 랜딩/배터리/뇌탭 모든 페이지 다크모드 대응

- ✅ `prefers-reduced-motion` 접근성 지원 (애니메이션 비활성화)
