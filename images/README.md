# 📸 사진 파일 준비 가이드

## 사진 배치하기

이 폴더에 다음 3장의 사진을 넣어주세요:

### 필요한 파일:
1. **photo1.jpg** - 첫 번째 추억 사진
2. **photo2.jpg** - 두 번째 추억 사진
3. **photo3.jpg** - 세 번째 추억 사진

## 📁 사진 준비 단계

### 1단계: 사진 선택
업로드하신 3장의 커플 사진을:
- 실내 사진 → `photo1.jpg`
- 야외 사진 → `photo2.jpg`
- 또 다른 사진 → `photo3.jpg`

### 2단계: 파일 복사
선택한 사진들을 이 폴더(`images/`)에 복사하고 위 이름으로 변경하세요.

### 3단계: 최적화 (선택사항)
더 빠른 로딩을 원하시면:
1. [TinyJPG](https://tinyjpg.com/) 방문
2. 3장의 사진 업로드
3. 압축된 파일 다운로드
4. 이 폴더에 저장

## ✅ 체크리스트

- [ ] photo1.jpg 파일 있음
- [ ] photo2.jpg 파일 있음
- [ ] photo3.jpg 파일 있음
- [ ] 파일 크기 확인 (각 5MB 이하 권장)
- [ ] 파일명 정확히 확인 (대소문자 구분!)

## 💡 팁

- **파일 형식**: JPG 또는 JPEG (PNG도 가능하지만 용량 큼)
- **권장 크기**: 1200px × 1600px (세로가 더 긴 사진)
- **용량**: 각 1-2MB 정도가 적당

## 🎨 사진 선택 가이드

좋은 사진 선택을:
- 밝고 선명한 사진
- 두 사람이 잘 나온 사진
- 행복한 순간을 담은 사진
- 의미 있는 장소에서 찍은 사진
